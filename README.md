Insurance Charges Analysis

Description: Analyzing how age, sex, BMI, children, smoking, and region affect insurance charges using Python and Power BI.

Dataset Columns:

age, sex, bmi, children, smoker, region, charges

Key Insights:

Smokers pay more.

Higher BMI → higher charges.

Older age → higher charges.

Minor regional differences.

Tools: Python (Pandas, Seaborn/Matplotlib), Power BI

Author: Nikam Snehal Anil
